# AfricArtz Admin Dashboard

Admin dashboard for internal platform operations.

## Development

Requirements:
- Node.js 18+
- npm

Install:
```bash
npm install
```

Run locally:
```bash
npm run dev
```

Useful commands:
```bash
npm run dev
npm run dev:prod
npm run build
npm run build:prod
npm run clean
```

## Deployment

Deployment and environment-specific details are intentionally kept out of the tracked README.

Use local environment files for development and your deployment platform configuration for hosted environments.
